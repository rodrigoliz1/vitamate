# Roadmap del Producto

## Fase 0 y 1: Definición, Arquitectura y Foundations
- Setup de Monorepo.
- Documentación y ADRs.
- Creación de estructuras UI (Ionic, Vite, Tailwind/CSS Variables).
- APIs base y configuración de CI/CD.

## Fase 2: Autenticación, Usuarios y Nutrición MVP
- Supabase Auth en Frontend y Backend.
- Flujo de Onboarding.
- Tablas base de Perfil.
- Diario de comidas manual y catálogo básico de alimentos.

## Fase 3: Motor de Entrenamiento e Historial
- Catálogo de Ejercicios.
- Planes de entrenamiento por motor determinista.
- Registro de Sesiones, Sets, y Progresiones (PRs).

## Fase 4: Inteligencia Artificial (Coach & Visión)
- Integración de LLMs (Chat Streaming).
- Análisis de fotos de comida para estimación.
- Asistente contextual con acceso (lectura) a planes.

## Fase 5: Suscripciones y Lanzamiento PWA
- Integración de Stripe (Mensual/Anual/Portal).
- Rate limits.
- Despliegue en Producción y Dominios.

## Fase 6: Híbrido (App Store / Google Play)
- Capacitor native builds.
- Migración a StoreKit/Google Play Billing.
- Integración con Apple Health.
