# VITAMATE

Monorepositorio para la plataforma VITAMATE (App, Website, API).

## Requisitos

- Node.js >= 18
- pnpm >= 8

## Instalación

```bash
pnpm install
```

## Desarrollo

Para arrancar todas las aplicaciones en paralelo:

```bash
pnpm dev
```

## Scripts Disponibles

- `pnpm build`: Compila todas las aplicaciones y paquetes.
- `pnpm lint`: Ejecuta el linter en todos los proyectos.
- `pnpm typecheck`: Verifica los tipos de TypeScript.
- `pnpm test`: Ejecuta las pruebas unitarias.
